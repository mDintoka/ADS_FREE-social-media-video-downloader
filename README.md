# 🚀 Ad-Free Social Media Video Downloader

## 📌 Overview

This is a simple **social media video downloader** built to solve a common problem:

> Most online downloaders are full of ads, redirects, and slow performance.

This tool provides a **clean, fast, and ad-free way** to download videos directly.

---

## ⚙️ Features

* 🚫 No ads
* ⚡ Fast downloading
* 🔗 Download videos using link
* 🎯 Simple and lightweight

---

## 🛠️ Requirements

* Python installed
* Git installed (optional)

---

## 📥 Installation

### 🔹 Method 1: Using Git

```bash
git clone https://github.com/mDintoka/ADD_FREE-social-media-video-downloader.git
cd ADD_FREE-social-media-video-downloader
social-media-video-downloader.py
```

---

## ▶️ Usage

* Run the script
* Paste video link
* Download without ads
